# Roblox Assistant skills

This folder holds the skill definitions that Assistant uses in Roblox Studio.

- `convert-to-streaming` converts a game to use instance streaming by enabling the setting, setting sensible defaults, refactoring code, validating client inputs, moving instances, and more.
- `device-simulator` tests the UI for your game on various devices and at different orientations.
- `docs-search` checks the Engine API reference documentation to ensure Assistant is working off of the most up-to-date information available.
- `scene-analysis` pulls data from `SceneAnalysisService` in order to analyze and optimize game performance. It makes recommendations around performance, helps identify memory leaks, and more.
- `virtual-input` simulates mouse, keyboard, and touch input to automatically test UI interactions.

See `SKILL.md` within each folder for more information.
